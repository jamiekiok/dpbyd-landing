Proactive design for privacy

Data Privacy by Design (DPbD) is a set of principles which promotes a proactive approach to system security, an initiative under the General Data Protection Regulation with which Avanade is responsible to comply. To better comply to the new regulation and protect personal information of Avanadeâ€™s stakeholders, we created a standardized toolkit which our developers can use to ensure that the security and privacy safeguards are met in new assets or applications in development.

Key Features

  • Easy-to-digest questionnaire
  • Customized checklist
  • Simple dashboard
  • Project compliance at a glance
  • Highly intuitive

Project Goals

This project is for our Informatics 2020 Capstone course, in which we created a Data Privacy by Design tool for software developers to use. With the formulation of data privacy regulations, such as the GDPR and the CCPA, proactive approaches to privacy are essential to ensure that the data of users is protected at all steps of the development process.

Partnership

This project is sponsored by Avanade, who wants to take proactive action to protect user data privacy. Once the project term is over, the tool will be fully transferred to the hands of Avanade.
